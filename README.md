# 🛡️ ComplianceShield – Ethical Vulnerability Scanner

ComplianceShield is a modern, professional-grade **ethical vulnerability scanning platform** built with Streamlit.  
It integrates powerful tools like Nmap and Nuclei to deliver **real-time security insights, risk scoring, and compliance mapping**.

---

## 🚀 Features

- 🔍 **Nmap Integration** – Port scanning & service detection  
- ☢️ **Nuclei Integration** – CVE & template-based vulnerability scanning  
- 🌐 **HTTP Security Analysis** – Headers, TLS, and misconfiguration checks  
- 📊 **Risk Scoring Engine** – CVSS-style scoring (0–10 scale)  
- 📋 **Business Impact Analysis** – Real-world risk interpretation  
- 🧾 **Professional Reports** – Export results as CSV & PDF  
- 🎯 **Compliance Mapping** – NIST, ISO 27001, PCI-DSS  

---

## ⚙️ Tech Stack

- **Frontend & Backend:** Streamlit  
- **Scanning Engine:** Nmap, Nuclei  
- **Language:** Python 3  
- **Visualization:** Plotly  
- **Reporting:** FPDF
